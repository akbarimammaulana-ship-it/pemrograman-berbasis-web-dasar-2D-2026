Tentang Saya

Halo, nama saya Intan Nuraini mahasiswa yang saat ini sedang menempuh pendidikan di bidang Ilmu Komputer. Saya memiliki ketertarikan di dunia teknologi dan hal-hal yang berhubungan dengan pengembangan sistem.

Saat ini saya masih dalam tahap belajar dan memahami dasar-dasar web development, seperti HTML dan penggunaan Tailwind CSS untuk membuat tampilan website. Dari proses pembelajaran tersebut, saya mulai mengenal bagaimana cara membangun sebuah website sederhana. Selain belajar di perkuliahan, saya juga mencoba mengerjakan beberapa project kecil sebagai bagian dari latihan agar lebih memahami materi yang sudah dipelajari. Saya masih terus belajar dan berusaha mengembangkan kemampuan di bidang teknologi, khususnya web development. Kedepannya, saya ingin terus meningkatkan kemampuan saya agar bisa lebih memahami dan menguasai bidang ini secara bertahap
